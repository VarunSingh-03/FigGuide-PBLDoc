# Project Title: FitGuide

## Description
FitGuide is a fitness guiding website where it guides you according to your height, weight and body type. It also provides exercise routine and a proper diet plan.

## Features
- User Body assessment 
- Provides Exercise Routine
- Provides Diet plan

## Tech Stack
- React.js
- Tailwind CSS
- Gemini API

## Installation
```bash
npm install
npm run dev
```

## Usage
Can be used to get your desired body shape
